# Raccourcis clavier

Vigie se pilote en grande partie au clavier. Les raccourcis sont inactifs pendant la saisie dans un champ
de texte ou lorsqu'une fenêtre modale / le mode lecture est ouvert.

| Touche | Action |
|---|---|
| `j` ou `↓` | Article suivant (et l'ouvre) |
| `k` ou `↑` | Article précédent (et l'ouvre) |
| `o` | Ouvrir l'article sélectionné dans le navigateur |
| `m` | Basculer lu / non-lu sur l'article sélectionné |
| `s` | Ajouter / retirer des favoris |
| `r` | Actualiser toutes les sources |
| `/` | Placer le curseur dans la recherche |
| `Échap` | Quitter le mode lecture |

> Astuce : `j`/`k` suivent l'ordre de la liste filtrée affichée (recherche, filtres et langue inclus).
